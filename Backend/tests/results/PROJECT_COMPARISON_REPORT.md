## 🔬 Research Contribution Analysis

### Chameleon's Novel Contributions

1. **Threat-Calibrated PSO (TC-PSO)**
   - **Novel Equation 1**: Dynamic inertia weight scaling
     - `w(t) = w_base · max(σ_min, 1 - α · A(t))`
   - **Novel Equation 2**: Anomaly-weighted objective function
     - `F'(t) = F(t) · (1 + β · A(t))`
   - **Innovation**: First PSO variant to incorporate real-time threat intelligence

2. **Semantic Deception RRT (S-RRT)**
   - **Novel Equation 3**: Exponential pheromone weighting
     - `Δτ' = Δτ · exp(Ψ - 1)`
   - **Novel Equation 4**: Depth-decay multiplier
     - `P'_expand = P_expand · max(ε, 1 - d/d_max)`
   - **Innovation**: First RRT variant with semantic understanding and memory optimization
